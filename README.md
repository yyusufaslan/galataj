## About This Repository

GalataJ is a closed-source Java profiler that shows performance metrics directly in your IDE. This repository serves as the **public hub** for:

- **Bug Reports** — Found something broken? [Open an issue](../../issues/new)
- **Feature Requests** — Have an idea? [Let us know](../../issues/new)
- **Discussions** — Questions, tips, or general conversation? [Join the discussion](../../discussions)

## Reporting a Bug

When opening a bug report, please include:

1. **Environment** — OS, IDE (VS Code / IntelliJ), IDE version, Java version
2. **GalataJ version** — found in the extension/plugin settings
3. **Steps to reproduce** — what you did before the issue occurred
4. **Expected vs actual behavior** — what you expected and what happened instead
5. **Logs** (if applicable) — output from the GalataJ panel or developer console

> **Tip:** Run **GalataJ: Health Check** first — it may resolve common issues automatically.

## Quick Start

New to GalataJ? Get started in under a minute:

1. Install from [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=galataj.galataj) or search **"GalataJ"** in VS Code Extensions
2. Run your Java application
3. Click the GalataJ icon at the bottom-right of your screen, then click **Start Profiling**

For detailed setup and usage, visit the [Documentation](https://www.galataj.com/docs).
